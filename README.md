# Fake vs Real Classification – Kaggle Project

## Overview

This project was developed as a **group project for a Machine Learning course**.
The goal is to build a **supervised classification model** that predicts whether a sample is **fake or real** using labeled data from the Kaggle *"Looks Like It's Fake"* challenge.

## Dataset

The dataset contains labeled samples used to train and evaluate classification models.

* Labels: **fake / real**
* Source: Kaggle competition *"Looks Like It's Fake"*

## Methods

Several supervised machine learning models were tested and compared, including:

* Random Forest
* Gradient Boosting
* Support Vector Machine (SVM)
* Logistic Regression
* k-Nearest Neighbors
* Decision Tree
* Multi-Layer Perceptron (Neural Network)
* AdaBoost

Model performance was evaluated using **cross-validation**.
